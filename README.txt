Welcome to my portfolio website! This is a showcase of my work, skills, and projects. Feel free to explore and get to know more about my background and experiences.

Website Overview
This website serves as my personal portfolio where I present:

   About Me: A brief introduction about my academic and professional journey.
   Projects: A detailed showcase of my completed projects with descriptions, tools used, and links to repositories.
   Skills: A list of my technical skills including programming languages, tools, and technologies.
   Contact: A form or link to reach out to me for collaborations, networking, or inquiries.

Key Features
   Responsive Design: The website is fully responsive, offering an optimal user experience across devices.
   Simple Navigation: Easy to navigate with clear sections for each part of my portfolio.
   Interactive Elements: Project showcases include links to GitHub repositories and live demos where applicable.

Technologies Used

  JavaScript: Adding interactive elements and smooth transitions.
